# Game Wizard

**Game Wizard** — плагин для ONLYOFFICE, который добавляет коллекцию мини-игр прямо в офисный редактор. Не покидая документ, пользователь может просматривать каталог игр, устанавливать или удалять отдельные игры и запускать их в отдельном изменяемом по размеру окне — всё это в привычном интерфейсе ONLYOFFICE.

## Обзор

Game Wizard — менеджер игр в виде плагина для ONLYOFFICE (Редактор таблиц и другие приложения). Плагин предоставляет:

- **Каталог игр**, который может работать локально или синхронизироваться с удалённым репозиторием на GitHub.
- Удобный **UI на основе карточек** для просмотра, установки, обновления и удаления игр.
- **Модальный лаунчер**, открывающий каждую игру в отдельном изменяемом окне внутри ONLYOFFICE.
- Полную поддержку **светлой и тёмной темы** с автоматическим переключением вслед за темой редактора.
- Локализацию интерфейса на **русский** и **английский** языки.

## Требования

| Компонент | Версия / Примечания |
|-----------|---------------------|
| ONLYOFFICE Docs | 7.0 и выше (Desktop или Server) |
| Браузер | Любой современный браузер с поддержкой ES5 (Chrome, Firefox, Edge, Safari) |
| Доступ в интернет | Необязательно — требуется только для синхронизации удалённого каталога через GitHub |

> **Сторонние инструменты сборки и пакетные менеджеры не требуются.** Плагин является самодостаточным набором статических файлов HTML, CSS и JavaScript.

## Установка

### Способ 1 — Менеджер плагинов ONLYOFFICE (рекомендуется)

1. Откройте ONLYOFFICE Docs (Desktop или Web).
2. Перейдите в **Плагины → Менеджер плагинов**.
3. Найдите *Game Wizard* и нажмите **Установить**.

### Способ 2 — Ручная установка

1. Скачайте или клонируйте репозиторий:
   ```
   git clone https://github.com/r7-consult/game_wizard.git
   ```
2. Скопируйте всю папку в директорию плагинов ONLYOFFICE:
   - **Windows (Desktop):** `%LOCALAPPDATA%\ONLYOFFICE\DesktopEditors\sdkjs-plugins\`
   - **Linux (Desktop):** `~/.local/share/onlyoffice/desktopeditors/sdkjs-plugins/`
   - **macOS (Desktop):** `~/Library/Application Support/ONLYOFFICE/DesktopEditors/sdkjs-plugins/`
   - **Server (Docker / DEB / RPM):** `/var/www/onlyoffice/documentserver/sdkjs-plugins/`
3. Перезапустите ONLYOFFICE.
4. Откройте любую таблицу, перейдите в **Плагины** и выберите **Game Wizard**.

### Способ 3 — Архив `.plugin`

1. Запустите скрипт сборки:
   ```powershell
   .\scripts\build-standalone-module.ps1
   ```
2. Полученный файл `.plugin` можно установить через функцию *Загрузить* в Менеджере плагинов ONLYOFFICE.

Новые игры можно добавить в каталог без обновления самого плагина.

## Функциональные возможности

### Каталог игр

При запуске Game Wizard объединяет два каталога:

- **Локальный каталог** (`modules/local-catalog.json`) — игры, входящие в поставку плагина или добавленные вручную.
- **Удалённый каталог** — загружается из настроенного репозитория GitHub (`modules/catalog.json`, ветка `main`). Игры из удалённого каталога можно устанавливать по запросу.

Каждая запись каталога содержит: имя папки модуля, отображаемое название, версию, иконку, ограничения размера окна и режим запуска.

### Состояния модулей

| Состояние | Значение |
|-----------|----------|
| `installed` | Игра присутствует локально и готова к запуску. |
| `available` | Есть в удалённом каталоге, но не установлена локально. |
| `update-available` | В удалённом каталоге доступна более новая версия. |
| `local-only` | Присутствует локально, но отсутствует в удалённом каталоге. |

### Запуск игры

При нажатии на карточку установленной игры она открывается в **модальном окне плагина** (через `Asc.PluginWindow`). Если API ONLYOFFICE недоступен (например, при разработке), игра открывается в обычном всплывающем окне браузера.

### Режим Hosted OLE

Если Game Wizard встроен в таблицу в виде OLE-объекта, он может получить JSON-данные через `initData` и автоматически открыть указанный игровой модуль без взаимодействия с пользователем.

### Поддержка тем

Плагин считывает текущую тему ONLYOFFICE (`theme-light` / `theme-dark`) и применяет соответствующий CSS-класс к корневому элементу. Смена темы применяется в реальном времени через `onThemeChanged`.

### Локализация

Строки интерфейса хранятся в JSON-файлах в папке `translations/`. Поддерживаемые языки: `ru-RU`, `en-US`.

## Конфигурация

Настройки плагина задаются в файле `scripts/manager-config.js`:

```js
window.GameWizardConfig = {
  branding: {
    title: 'Game Wizard',
    subtitle: 'Выберите игру для запуска'
  },
  local: {
    modulesRoot: 'modules',
    bundledCatalogPath: 'modules/catalog.json',
    localCatalogPath: 'modules/local-catalog.json'
  },
  remote: {
    provider: 'github',
    repositoryUrl: 'https://github.com/r7-consult/game_wizard',
    branch: 'main',
    modulesRoot: 'modules'
  },
  hiddenModules: []   // идентификаторы модулей, скрытых из UI
};
```

## Структура проекта

```
game_wizard/
├── config.json                  # Манифест плагина ONLYOFFICE
├── index.html                   # Главная страница плагина
├── module-window-host.html      # Оболочка окна размещённой игры
├── modules/
│   ├── catalog.json             # Встроенный список удалённых модулей
│   ├── local-catalog.json       # Список локальных модулей
│   └── <game_id>/               # Папка каждого игрового модуля
├── resources/
│   ├── icons/                   # Иконки плагина (светлая/тёмная)
│   └── images/                  # Графические ресурсы (маскот и т.д.)
├── scripts/
│   ├── manager-config.js        # Конфигурация времени выполнения
│   ├── plugin.js                # Жизненный цикл плагина ONLYOFFICE
│   ├── module-service.js        # Управление каталогом и операции с модулями
│   ├── game-launcher.js         # Логика запуска игр
│   └── ui.js                    # Вспомогательные функции для DOM
├── styles/
│   ├── main.css                 # Стили оболочки плагина
│   └── game-shell.css           # Стили оболочки игры
├── translations/
│   ├── langs.json               # Поддерживаемые локали
│   ├── en-US.json               # Строки на английском
│   └── ru-RU.json               # Строки на русском
└── vendor/                      # Сторонние библиотеки
```
